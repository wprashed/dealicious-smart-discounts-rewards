== Description ==
Dealicious is a WooCommerce plugin that helps store owners engage customers and boost sales with smart discounts and reward features.

Features include:
* First-Time Buyer Discount – automatically apply a discount for new customers.
* Spin-to-Win Coupon Wheel – interactive wheel to reward customers with coupons.
* Cart-Based Discounts – flexible deals based on cart contents.
* Birthday Discount Emails – automatically send birthday coupons to users.
* Points for Reviews – reward users for leaving product reviews.

== Installation ==
1. Upload the `dealicious-smart-discounts-rewards` plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. To display the Spin-to-Win wheel on any page, add the shortcode: `[dealicious_spin_wheel]`.
4. Configure additional settings in WooCommerce if needed.

== Frequently Asked Questions ==
= Can I customize the coupon amounts and codes? =
Currently, the first-time discount uses the coupon code `FIRSTTIME10` with 10% off. Future updates may add admin settings for customization.

= Does it support multiple languages? =
Yes, the plugin is translation-ready. Place `.mo` files in `/languages/` or use a translation plugin.
