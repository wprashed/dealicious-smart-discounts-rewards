== Description ==
Smart discounts and rewards plugin for WooCommerce stores.

* First-Time Buyer Discount
* Spin-to-Win Coupon Wheel
* Cart-Based Discount (extendable)
* Birthday Discount Emails
* Points for Reviews

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via WordPress > Plugins
3. Use shortcode `[dealicious_spin_wheel]` to show the wheel
